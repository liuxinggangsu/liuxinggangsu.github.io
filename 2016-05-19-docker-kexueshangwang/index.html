Docker（灵雀云）2016最新免费搭建科学上网服务

发表于 2016-05-19   |   分类于 WEB开发   |     |   本文阅读量 次

科学上网服务可以说是捣鼓网络工作者基本都会接触到的东西，因为伟大的google迁出中国而百度的搜索结果让人很难满意，使用google搜索可以说经常的事情，虽然每天的频次不高，所以如何使用google搜索，一直在研究，从开始的修改host文件，到被一个技术男引导搭建vps上shadowsocks服务，虽说vps不贵，当初的初心也是体验一下国外vps的特色，在512m内存的vps搭建了本博客，但vps的续费也是让我犹豫不决的，所以免费翻墙功夫还得研究，加上工作中本地网络对某些网站是过滤还是什么，就是在本地打不开，在其他网络上没有问题，这个问题一直很纠结，老板感觉是网站出问题了，其实是网络的事，但这个事就有点和管网络的同事解释不清，现在想到的办法就是用代理ip浏览网站，但是网上发布的很多代理ip都是不管用，一个个试很费功夫，还很快ip就不能使用了。

今天不经意间看到一篇用docker搭建免费shadowsocks的文章，就按博主所说一步步尝试了一下，发现香港的服务器已经停止使用，网上说的方法都是去年的如今已经不管用了，提示充值，百度了一下发现

2015-12-02 香港一区的服务已经全部停止。

消息称，香港一区不再免费。
于是百度了一上午发现很多文章还是去年的，就尝试的用灵雀云官方说的绑定官方微信号免费领取优惠券的方法搭建成功了，不过好像需要每个月手动去领取优惠券，否则会被停掉！

原理

其实就是借用了蓝灯的代理服务，在Docker搭了一个转发。

优点就是你的电脑不用安装蓝灯了，且能支持手机使用！！

缺点蓝灯的速度有时快，有时慢，不稳定。

准备工作

  1. 注册账号，点击这里（含我的推广信息，介意的请百度 灵雀云） 前往注册。注册需要验证手机号和邮箱。
  2. 灵雀云新政策需要绑定微信才可以领取优惠券(该优惠券实现免费使用Docker，具体上官网自己研究不多说)。
  3. Docker镜像 index.alauda.cn/zhaxg/docker-lantern-ss (不懂，可忽略)。
  4. (可选) SwitchySharp扩展

部署服务

  1. 访问：https://hub.alauda.cn/ 搜索 docker-lantern-ss
    搜索结果如图

当然选择最上面那个收藏下载多的，点击进去后，点击创建服务

填写上服务名称，以前教程容器实例大小选择256m或者512m的，因为以前这两个香港服务器是免费的，我开始也是选择这两个中一个，创建好之后我又更新成4g的那个竟然也是可以的。

其他的下一步按默认配置就可以，最好修改一下环境变量配置，不修改默认密码是password

把 SS_PASSWORD 可以修改成自己想要的密码，其他的按表格填就行了。

点击【创建服务】。

经过一段时间的等待，显示如下图，说明部署完成。

点击插卡 【日志】 显示下图。说明 Shadowsocks服务部署好了。

如果，日志中无显示内容，或者一直处于 部署中.. 建议重新部署检查配置内容。

这样就部署成功了，可以在本地安装个ss客户端就是个exe文件，点击下载ss客户端，

For Windows 7 or earlier, download shadowsocks-win-2.3.zip

For Windows 8 or later, download shadowsocks-win-dotnet4.0-2.3.zip
tcp:// 后面部分就是服务器地址和端口

密码就是 配置环境变量中的密码。示例中是 NIUXSS

加密方式是配置环境变量中的方式。示例中是 AES-256-CFB

如何嫌麻烦可以使用Proxy SwitchySharp插件，可以轻松管理和切换代理设置，对于代理的设置一定别忘了选择对所有协议均使用相同服务器，刚开始我没有选择发现本地ip没有变化，就想到最近很多网站都支持https了，所以看了下https代理没有设置，选择所有协议使用相同设置后ip就换成代理ip了

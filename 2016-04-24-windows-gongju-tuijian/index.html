用对的工具做对的事——我的Windows软件

发表于 2016-04-24   |   分类于 转载日志   |     |   本文阅读量 次

整理一下自己多年来搜集的常用PC软件，Windows平台，有装机必备，也有些不常用但是找不到就会抓狂的神器，附上官方链接方便使用。

网盘

  • 坚果云
    最好的网盘工具，免费版每月可上传1G，下载3G。多文件夹可以分别同步，支持增量同步。服务器在国内，速度快，稳定。
  • Onedrive
    微软家的网盘，与Outlook邮件配合较好，网页版国内无法访问，客户端不受影响。
  • BitTorrent Sync
    P2P多终端文件同步，不经过云服务器。文件最终存储在用户某台机器的硬盘里。
    云同步的便利无需多言，使用哪家网盘服务更是重要。不要使用百度网盘保存重要文件，原因你懂的。当然Dropbox、Google Drive也是靠谱的。

网络代理

  • Shadowsocks
    必备的上网代理工具。
  • 蓝灯
    备用，免费。
  • 赛风
    备用，免费。
    科学上网必备，享受自由互联网。

密码管理

  • KeePass
    免费开源密码管理软件，配合靠谱的网盘工具实现随时随地查看密码。

浏览器

  • Google Chrome
    如果只能用一款浏览器，那就是Chrome了，设置、扩展、浏览历史、网站密码全部同步。

云笔记

  • 为知笔记
    个人知识管理工具，收集灵感，随时随地记录想法。同类型的还有OneNote、Evernote、有道云笔记，各家差距不大，适合就好，重在回顾整理。

输入法

  • QQ输入法
    Windows10下表现良好的输入法，微软自带拼音输入法修改输错的拼音极其反人类！不要问我搜狗好不好，很久不用了，输入法毕竟只是工具，找到适合自己的，别乱弹窗整无用的功能，就好了。

压缩软件

  • Bandizip
    韩国人开发的压缩工具，可以直接看压缩包内的图片是一大特点。
  • 7z
    高效、免费的压缩工具，虽然界面丑了点。

电子书管理

  • Calibre
    整理控必备，支持Epub、Mobi等多种格式转化，Kindle用户必备。

办公&字处理

  • Microsoft Office
    Word、PowerPoint、Excel、Visio，都是神器。谁敢说真正掌握Office技能？
  • Dia
    免费开源的流程图绘制软件。
  • Notepad ++
    文本编辑器，偶尔看看代码还是很方便的。（必装compare、explorer插件）

PDF

  • Adobe Acrobat Reader
    最靠谱的PDF阅读器，唯一缺点就是体积太大。
  • SumatraPDF
    免费小巧的PDF阅读器，还支持Epub、Mobi、CHM等格式。
  • PDFill PDF Tools
    免费的PDF编辑工具，支持PDF合并、加密、加水印等简单编辑。
  • Adobe Acrobat
    专业的PDF编辑工具，收费。
  • doPDF
    免费的PDF打印机，可以用来转化许多格式的文件。

音乐

  • Foobar2000
    听本地音乐的最佳选择，一般使用Asion汉化的版本，附带了一些有用的插件。支持免安装，利用网盘同步即可。
  • 网易云音乐
    网易大法好！虽然是云服务，但有些功能比如私人FM只有客户端才有。

即时通讯

  • QQ/TM
    大天朝网民最离不开的工具，个人感觉其实比微信好用。官方有轻聊版，可惜现在Windows版要的权限也多了起来，还会开启后台服务。

视频

  • QQ影音
    本地播放器的最简单选择，使用方便，支持多种解码器，多字幕，在线搜字幕。
  • Potplayer
    进阶版的播放器，爱折腾党必备。

图片浏览

  • FastStone Image Viewer
    免费、功能强大、操作简便，支持gif、psd格式图片查看，简单的图片裁剪、放大缩小，还能图片无损拼接。

邮件客户端

  • Outlook
    微软家的，包含在Office365中。功能强大，与微软的邮箱搭配最好。
  • Foxmail
    国产客户端，简单易用。
    现在一般主要靠手机接收邮件，PC上主要用网页版邮箱，客户端还有Thunderbird、YoMail可供选择。

下载工具

  • Free Download Manager
    免费、开源下载工具，支持BT。
  • QQ旋风
    国产，还算良心，不会像迅雷一样屏蔽某些磁力链接。

词典翻译

  • 灵格斯词霸/有道词典/欧陆词典

设计创意

  • Adobe全家
    Photoshop、Illustrator、Premiere、After Effect、Audition这几个牛逼的软件就不多说了，真正能用好一个就不愁吃饭了。全球光靠卖软件能赚大钱的公司也没几个了，等有钱了一定要用上正版Adobe！
  • Aegisub
    免费跨平台的字幕编辑工具。
  • Audacity
    免费、开源、跨平台的音频编辑软件，可替代Audition部分功能。
  • GIMP
    免费、开源的图像处理软件，部分功能类似Photoshop。
  • FreeMind/Xmind/MindManager
    思维导图工具，FreeMind开源免费。
  • Axure
    原型制作。

文件浏览器

  • Total Commander
    功能强大的文件管理器，可以批量重命名文件、压缩文件、比较同步文件夹……（配合插件：office预览、quickSearch eXtended、txQiuckView预览插件、ulister、excellence(excel预览)）
    类似的软件还有FreeCommander。

屏幕录制

  • Apowersoft 免費線上螢幕錄影
    台湾产的屏幕录制工具，曾经用它录制过1个多小时的网络直播课程。
  • CamStudio
    免费、开源的屏幕录制工具，貌似还能模拟声卡输出。类似的还有oCam，FSCapture也能录制屏幕。

其他

  • ABBYY FineReader
    最好的OCR识别软件，收费。
  • Unified Remote
    把手机变成电脑的遥控器，通过Wi-Fi通信控制电脑。支持Windows、iOS等多平台。
  • FSCapture
    截屏工具，收费，可以滚动截屏、简单标记，甚至还能录制屏幕。
    同类型软件还有PicPick，免费开源的Greenshot。
  • VirtualBox
    免费的虚拟机软件。
  • Bluestack
    安卓虚拟机。
  • Unlocker
    每次弹出U盘时碰到“有程序正在占用文件”时就会抓狂，这时候就可以用这个工具来解决。
  • RBTray
    右击即可最小化任何程序到桌面右下托盘（上班防领导神器XD）
  • FileZilla
    免费、开源、专业的FTP软件。
  • MorphVOX Pro
    变声工具。WS男网聊神器 所以千万不能相信网上只有声音好听的妹子！
  • 维棠
    Flv视频下载。
  • Ditto
    剪贴板增强工具。免费，开源。
  • Total uninstall
    完全卸载，专业卸载工具，会监控安装软件后系统注册表、文件系统等发生的变化，可以回滚。
  • ExifTool
    元数据查看工具。
  • Image MetaData Stripper
    图片元数据清除工具。

最后要说的

  1. 一定要到官方网站去下载软件！不要在奇怪的网站下载软件，定期更新软件。
  2. Windows7以后可以不安装杀毒软件，系统自带杀软。要另外安装的话，可以试试Avast、小红伞Avira 等国外软件。
  3. 拒绝搜狗拼音、360、暴风影音等流氓软件，尽量使用正版软件。许多人宁愿用着盗版的WinRar，每次打开看到授权到期也不去尝试使用7z、Bandizip这些免费好用的替代品。
  4. 用对的工具做对的事。很多软件没有好坏之分，都只是工具而已。不必太纠结，比较之后选择适合自己的就好了。如何使用好工具完成事情才是关键。日本74岁的老爷爷还能用Excel画出精致的风景画呢！
  5. 尽量多用云服务吧。现在几乎很少有人会特意去下载流行歌曲或视频到电脑里，都是在线享受。随着网络带宽提升、技术升级，纯本地客户端软件只会越来越少，云服务是趋势。相信在未来某一天，所有的软件、应用服务都会基于网络，而我们需要的只是一个账号，在任何地方登陆互联网，都能享受到一样的服务。

记一次投票刷票经历222

发表于 2017-01-14   |   分类于 慢生活   |     |   本文阅读量 次

对于一般的投票都会限制ip投票次数，虽然以前也接触过代理ip啥的，但很多代理ip都是不可用的，关键是操作起来太麻烦，无意间找到了SQSX免费代理，当时是由于一个关键词优化在百度站长平台中看关键词排名已经挺好了，在本地打开就是排名很靠后，网上说的一般是百度服务器没有更新，我就想测试下通过chinaz工具测试到的在外省排名是否属实，就用了一下这个代理工具，选择全局代理，在百度一下ip就可以看到本地ip已经换了。
正好最近参加了一次投票评选，看着别人的票刷刷的涨，似乎感觉人家也在刷票，主要通过朋友圈啥的拉票，效果是非常一般的，一般人家投一票就敷衍过去了，想要排名是很难的，正好有上面那套软件，可以用山东的一些ip去刷，于是就去实施了一下，那个投票网站目的是防止别人刷票咋的，需要投好几个项目的票，每个项目还得必须投5票，由于每个项目选手很多，网站本身做的毫无用户体验，自己手工点击投票是比较费劲，于是就从网上找了些刷票软件，可以说基本上都不可以用，但每个我都试了一下，恰好一个刷票软件有个模拟啥的，虽然不能用，让我把思路引导了模拟投票上了，既然自己投票太费劲，那就让电脑自己自动操作，于是找到了蓝欣投票软件自助投票器，自动模拟投票过程，到输入验证码时按f4停止，手工输入验证码提交。由于这次投票名次不是关键，我也没有用这个办法去用力操作，这里主要是记录一下刷票的思路，对于像微信投票可以微信多开啥的，这样的方法就是不用花钱，耗点时间。

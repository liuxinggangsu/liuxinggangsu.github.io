这样的爱情，谁不想拥有？

发表于 2016-03-02   |   分类于 我家大小姐   |     |   本文阅读量 次




我们的小窝，不用很大，

充满笑声，温馨就好。



我们的存款，不用很多，

父母老了，我们负担得起就好。



老公，不用很帅，

风雨一生，足够的包容和担当就好。



孩子，不用很优秀，

平安长大，懂得如何做人就好。



生活，不求一帆风顺，

磕磕绊绊，我们能共同面对就好。



工资，不求多高，

偶尔出游，偶尔庆祝，够开销就好。



当我们老去，回看这一生，

没有太多遗憾就好。

笑看风雨，平安知足，

步履蹒跚，有你，有我，有孩子，

仅仅如此，就好。






PS：这几天老婆一直在娘家，虽然下班不用看孩子，但这几天感觉是非常的忙碌，上次老婆回娘家住的时间不短，由于与老妈的小摩擦，这次我也没有能很好的化解她俩之间的矛盾，加上家里冷，虽然我骨子里认为温室里培养出的花朵经不起风雨，这样的想法似乎被老婆冠以不爱女孩的表现，其实我是骨子里不重男轻女的，有些东西做的并不一定是内心想的，前些天我想让老婆回家，她说让老爸多歇歇，突然我感觉我对老婆、老爸、妈的关心真的太少了，我总以上班的名义名正言顺的偷懒，老婆被老妈执拗的认为在家看孩子就的看一天，那次我星期天在家看了一天，比上班累多了，但我从心里缺少对老婆的关心，老婆经常在娃午睡时偷偷回屋开电脑看电视剧，我也没想家里电视好久都没台了，也许我多多的为老婆想一想，她对老妈的戾气就会被我的温馨所感化，她没有从我这里获得温暖，在老妈的“唆使”下老爸对老婆的态度也有所变化，让本就小心眼的老婆感到伤心落泪，我夹在中间，无所谓的态度让两边都伤心，工作就是工作，不要影响了生活，多关心家人吧！！！

立马买个网络机顶盒，让老婆肥皂剧看个够！！！

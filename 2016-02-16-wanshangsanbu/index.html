晚上散步随想

发表于 2016-02-16   |   分类于 慢生活   |     |   本文阅读量 次

今天吃过晚饭突然心血来潮想出去散散步，也许是每周一天的休班，而休班往往也是比较忙，始终感觉生活的节奏很忙碌，一码一码的很紧凑，也许是娃的到来让自己的空间变得越来越小，每天也不知道自己忙的啥就是感觉特忙，忙的脑子里面就是睡觉的时候也静不下心来，今天晚上突然遛遛弯，感觉心情豁然开朗一些，有时候人就的静下来回头看看当初的起点目的是什么，现在是不是和当初的思想越走越远。

今天还是大年初九，明显感觉年已经远去，用我老妈念叨的话说，过了大年三十初一，半年来霍霍的年就过去了，现在听的比较多看到的比较多的也是越来越没有年味了，走在本应该很熟悉的村子里早就没有零星年味，就像春晚中大圣没有归来，也许是经济危机，也许是自己上班时不曾留意，走过曾经的初中校园，看到了很多店铺改了门庭，很火爆的烧烤一条街也被蒙古包所包裹，烧烤的工具也都一致的换了新的，记得没几天万家乐超市旁边的一个火锅店还要转让来，今天已经改头换面成新的火锅店了，一路走来，在这个经济危机的时刻，突然就想哪些店铺能持久的活了下来，像百货大楼、万家乐超市这样的老牌传统行业门面日复一日也不需要过多的宣传，固有的人群就活的如日中天，信息的不对称药品行业的暴利，感觉多了不少药店连锁店，是药店的开店门槛放低了，还是昌平花园带动了村里的经济，不得而知，像安齿口腔，我上初中就从那里看过牙，如今父子两代人的坚持，也赶上了口腔行业的黄金时代，如今门面从一两间变成一大排的大店，像滋仁轩由于老婆在那里上过班，就靠着所谓的药方也活的非常滋润，固定客户也是源源不断，虽然有时候就是病人的自欺欺人，门头最多的就是饭店了，门面大种类众多，还有就是理发店也非常的多，记得以前理个发也就一块钱，物价虽然上涨了非常多，但感觉理发的成本没有增加多少，像我们这种城边上理个发也的五六块钱了，据说城里都十五二十的。
